Meta Tags Explained

<meta> tags provide metadata about an HTML document. Metadata is information about the document that is not displayed directly on the page but is used by browsers and search engines. Here is a breakdown of the <meta> tag and its attributes:

Attributes

  • name: Specifies the name of the metadata. This attribute is used to provide specific information, such as the description or author of the document.
  • content: Provides the value associated with the name attribute. For instance, it might contain a description of the page's content or the name of the author.
  • http-equiv: Provides an HTTP header for the information/value of the content attribute. This is used for setting values that would normally be sent via HTTP headers, such as Content-Type or Cache-Control.
  • charset: Specifies the character encoding for the HTML document. For example, UTF-8 is a common character encoding that supports most characters and symbols used in various languages.
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="A sample page demonstrating HTML meta tags.">
<meta name="author" content="Your Name">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="robots" content="index, follow">
<meta name="keywords" content="HTML, meta tags, example">
