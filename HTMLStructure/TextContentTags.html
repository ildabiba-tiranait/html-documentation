HTML Text Content Tags Explained

<h1> to <h6> Tags

The <h1> to <h6> tags define HTML headings, with <h1> being the highest level and <h6> being the lowest. They are used to structure the content hierarchically and improve readability and SEO.

Attributes: These tags share common attributes:

  • id: Provides a unique identifier for the heading.
  • class: Assigns one or more class names for styling purposes.
  • style: Applies inline CSS styles directly to the heading.
  • title: Offers additional information about the heading, often displayed as a tooltip.
<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Sub-sub Heading</h3>

<q> Tag

The <q> element indicates a short inline quotation. It is used to quote a short excerpt from another source. Browsers typically render it with quotation marks around the text.

Attributes:

  • cite: Contains a URL to the source of the quotation.
  • id: Provides a unique identifier for the quotation.
  • class: Assigns one or more class names for styling purposes.
  • style: Applies inline CSS styles directly to the quotation.
  • title: Offers additional information about the quotation, often displayed as a tooltip.
<p>He said, <q cite="http://example.com">Hello, world!</q></p>

<time> Tag

The <time> element defines a specific time or datetime. It is used to represent dates and times in a machine-readable format, often used for scheduling or timestamps.

  • class: Assigns one or more class names for styling purposes.
  • style: Applies inline CSS styles directly to the time element.
  • title: Offers additional information about the time element, often displayed as a tooltip.
<p>The event will take place on <time datetime="2024-08-15">August 15, 2024</time>.</p>
