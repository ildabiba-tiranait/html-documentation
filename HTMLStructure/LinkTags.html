Link Tags: <a> and <link>

<a>

The <a> element, also known as the anchor element, is used to define a hyperlink that links one page to another. It is one of the most important HTML elements for creating navigable web pages.

The <a> element can link to external resources, internal pages, or even trigger actions such as downloading files. It is versatile and supports various attributes to control its behavior and appearance.

Attributes:
  • href: Specifies the URL of the page the link goes to. This can be an absolute URL (e.g., https://www.example.com) or a relative URL (e.g., page.html).
  • target: Specifies where to open the linked document. Values can include:
    • _self: Opens the link in the same frame or window (default).
    • _blank: Opens the link in a new window or tab.
    • _parent: Opens the link in the parent frame.
    • _top: Opens the link in the full window.
  • download: Specifies that the target will be downloaded when a user clicks on the hyperlink. It can also specify a name for the file to be downloaded.
  • rel: Specifies the relationship between the current document and the linked document. Common values include noopener, noreferrer, and nofollow.
  • hreflang: Specifies the language of the linked document. Useful for linking to documents in different languages.
  • type: Specifies the media type (MIME type) of the linked document. For example, text/html for HTML documents.
  • id: Provides a unique identifier for the <a> element, useful for targeting with CSS or JavaScript.
  • class: Assigns one or more class names to the <a> element for styling with CSS.
  • style: Applies inline CSS styles directly to the <a> element.
  • title: Provides additional information about the <a> element, usually displayed as a tooltip when the user hovers over the link.
<a href="https://www.example.com" target="_blank" download="file.pdf" rel="noopener" hreflang="en" type="application/pdf" id="exampleLink" class="link" style="color: blue;" title="Visit Example.com">Visit Example</a>

<link>

The <link> element is used to define the relationship between a document and an external resource. It is commonly used in the <head> section of an HTML document to link to stylesheets or other external resources.

The <link> element does not have closing tags and is self-contained. It is essential for linking CSS files to style the HTML document.

Attributes:
  • href: Specifies the URL of the linked document. For example, the URL of a CSS file.
  • rel: Specifies the relationship between the current document and the linked document. Common values include stylesheet for linking CSS files and icon for defining a favicon.
  • type: Specifies the media type (MIME type) of the linked document. For example, text/css for CSS files.
  • media: Specifies the media type for which the linked resource is designed. For example, screen for screen displays or print for print styles.
  • hreflang: Specifies the language of the linked document. Useful for linking to stylesheets for different languages.
  • sizes: Specifies the size of the linked resource, often used with icons to define their sizes.
  • id: Provides a unique identifier for the <link> element, useful for targeting with CSS or JavaScript.
  • class: Assigns one or more class names to the <link> element for styling with CSS.
  • style: Applies inline CSS styles directly to the <link> element.
  • title: Provides additional information about the <link> element, usually displayed as a tooltip when the user hovers over the link.
<link href="styles.css" rel="stylesheet" type="text/css" media="screen" hreflang="en" sizes="16x16" id="stylesheetLink" class="linkStyles" style="display: none;" title="Main Stylesheet">
