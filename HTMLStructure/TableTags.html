HTML Table Tags Explained

<table> Tag

The <table> element defines a table in an HTML document. Tables are used to present data in a structured format, with rows and columns.

Attributes:

  • border: Specifies the width of the border around the table. For example, border="1" creates a 1-pixel border.
  • cellpadding: Specifies the space between the cell wall and the cell content. This is measured in pixels.
  • cellspacing: Specifies the space between cells. This is also measured in pixels.
  • summary: Provides a summary of the table's purpose and structure, which is helpful for screen readers and accessibility.
  • id: Provides a unique identifier for the table element.
  • class: Assigns one or more class names for styling purposes.
  • style: Applies inline CSS styles directly to the table element.
  • title: Offers additional information about the table element, often displayed as a tooltip.
<table border="1" cellpadding="5" cellspacing="0">
    <caption>Sample Table</caption>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="2">Footer</td>
        </tr>
    </tfoot>
</table>

<caption> Tag

The <caption> element defines a caption for a table. It is used to provide a title or description for the table, making it easier to understand the table's content.

Attributes: Similar to the <table> tag.

<table>
    <caption>Employee Details</caption>
    <tr>
        <th>Name</th>
        <th>Position</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>Manager</td>
    </tr>
</table>

<thead> Tag

The <thead> element groups the header content in a table. It is used to define a set of rows that contain column headers, which are often repeated at the top of the table.

Attributes: Similar to the <table> tag.

<table>
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Alice</td>
            <td>30</td>
        </tr>
    </tbody>
</table>

<tbody> Tag

The <tbody> element groups the body content in a table. It contains the main data rows of the table and is typically used to separate these rows from the header and footer rows.

Attributes: Similar to the <table> tag.

<table>
    <thead>
        <tr>
            <th>Product</th>
            <th>Price</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Laptop</td>
            <td>$1000</td>
        </tr>
    </tbody>
</table>

<tfoot> Tag

The <tfoot> element groups the footer content in a table. It is used to define a set of rows that typically contain summary information or footnotes. The <tfoot> element is often placed after the <tbody> but is rendered before it, which allows the footer to be visible during scrolling in long tables.

Attributes: Similar to the <table> tag.

<table>
    <tfoot>
        <tr>
            <td colspan="2">Total</td>
            <td>$2000</td>
        </tr>
    </tfoot>
    <tbody>
        <tr>
            <td>Item 1</td>
            <td>$500</td>
        </tr>
    </tbody>
</table>

<tr> Tag

The <tr> element defines a row in a table. It is used to group together table cells (<td> or <th>) that belong to the same row.

Attributes: Similar to the <table> tag.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

<th> Tag

The <th> element defines a header cell in a table. Header cells are used to label the columns or rows in a table and are typically displayed with bold text and centered alignment by default.

Attributes:

  • colspan: Specifies the number of columns a header cell should span.
  • rowspan: Specifies the number of rows a header cell should span.
  • scope: Specifies whether the header cell is for a column, row, or a group of columns or rows.
  • abbr: Provides an abbreviated version of the content in a header cell, useful for accessibility.
  • id: Provides a unique identifier for the header cell.
  • class: Assigns one or more class names for styling purposes.
  • style: Applies inline CSS styles directly to the header cell.
  • title: Offers additional information about the header cell, often displayed as a tooltip.
<table>
    <tr>
        <th colspan="2">Header</th>
    </tr>
    <tr>
        <td>Cell 1</td>
        <td>Cell 2</td>
    </tr>
</table>

<td> Tag

The <td> element defines a cell in a table. It is used to hold data in a table and can be used within rows defined by the <tr> element.

Attributes:

  • colspan: Specifies the number of columns a cell should span.
  • rowspan: Specifies the number of rows a cell should span.
  • headers: Specifies the header cells that apply to a cell, used to provide additional context for accessibility.
  • id: Provides a unique identifier for the cell.
  • class: Assigns one or more class names for styling purposes.
  • style: Applies inline CSS styles directly to the cell.
  • title: Offers additional information about the cell, often displayed as a tooltip.
<table>
    <tr>
        <td colspan="2">Spanned Cell</td>
    </tr>
    <tr>
        <td>Cell 1</td>
        <td>Cell 2</td>
    </tr>
</table>
