Interactive HTML Tags: <details> and <summary>

<details>

The <details> element is used to create a disclosure widget from which the user can obtain additional information or controls. It is often used to provide expandable content, such as a collapsible section of a page.

When a <details> element is clicked, it toggles between showing and hiding its content. This behavior is built into the browser and does not require JavaScript to function.

Attributes:
  • open: A boolean attribute that specifies whether the <details> element should be visible (open) when the page loads. If present, the details will be open by default. If omitted, the details will be closed initially.
  • id: Provides a unique identifier for the <details> element, useful for targeting with CSS or JavaScript.
  • class: Assigns one or more class names to the <details> element, allowing for custom styling with CSS.
  • style: Applies inline CSS styles directly to the <details> element.
  • title: Provides additional information about the <details> element, usually displayed as a tooltip when the user hovers over the element.
<details open>
  <summary>More Information</summary>
  <p>Here is some additional content that can be expanded or collapsed.</p>
</details>

<summary>

The <summary> element is used within a <details> element to define a visible heading for the disclosure widget. The content of the <summary> is always visible, and clicking it will toggle the visibility of the content within the <details> element.

The <summary> element provides a way for users to see a brief description or title of the expandable content, making it clear what additional information or controls are available.

Attributes:
  • id: Provides a unique identifier for the <summary> element, which can be used for targeting with CSS or JavaScript.
  • class: Assigns one or more class names to the <summary> element for custom styling with CSS.
  • style: Applies inline CSS styles directly to the <summary> element.
  • title: Provides additional information about the <summary> element, typically displayed as a tooltip when the user hovers over the element.
<details>
  <summary>Click to Expand</summary>
  <p>This is the hidden content that becomes visible when the <summary> is clicked.</p>
</details>
